📘 GRAMMAR FORMULAS · UNITS 1–3

🔹 be: I / You

  • I'm + nationality/name/job
  • You're + adjective/nationality
  • Are you + ... ? → Yes, I am. / No, I'm not.

🔹 be: he/she/they

  • He's / She's + (age/job/adjective)
  • They're + my/your/his ...
  • Is he/she + ... ? → Yes, he is. / No, she isn't.

🔹 Possessives

  • my/your/his/her/its/our/their + noun
  • name's + noun (Maria's book)

🔹 There is / There are

  • There's + a/an + singular noun
  • There are + (number) + plural noun

🔹 It is / This / These

  • It's + a/an + noun
  • This + singular / These + plural

🔹 Simple present

  • I like + noun / I don't like
  • Do you + verb? → Yes, I do. / No, I don't.
